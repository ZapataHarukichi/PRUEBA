Hill Climbing y Algoritmos Genéticos

Introducción

Hill Climbing y los Algoritmos Genéticos son dos técnicas de búsqueda.} Ambos buscan soluciones óptimas, pero lo hacen de formas distintas

¿Qué es Hill Climbing?

Hill Climbing es un algoritmo de búsqueda heurística utilizado para encontrar la mejor solución posible (óptimo local) subiendo paso a paso hacia soluciones cada vez mejores.

Funciona empezando desde una solución inicial y luego se mueve a una solución vecina si esta es mejor. El proceso se repite hasta que no hay ninguna solución vecina mejor.

Ventajas:

  • Fácil de implementar
  • Rápido en problemas simples

Desventajas:

  • Puede quedarse atrapado en óptimas locales
  • No explora todo el espacio de soluciones

Problema del Viajante

Minimizar la distancia recorrida por el viajante al visitar varias ciudades.

¿Qué son los algoritmos genéticos?

Los Algoritmos Genéticos (AG) son métodos de optimización inspirados en la evolución biológica. Funcionan mediante una población de soluciones que evoluciona a través de operadores como selección, cruce y mutación.

Cada solución se representa como un "cromosoma" (generalmente una cadena de bits, números o caracteres). A lo largo de generaciones, las mejores soluciones se seleccionan para reproducirse y generar nuevas posibles soluciones.

Principales etapas de un AG:

Inicialización

Se genera una población aleatoria de soluciones.

Evaluación

Se mide qué tan buena es cada solución (fitness).

Selección

Se eligen las mejores soluciones para reproducirse.

Cruce

Se combinan dos soluciones para crear una nueva.

Mutación

Se modifica aleatoriamente una parte de la solución.

Repetición

El proceso se repite por varias generaciones.

Ventajas:

  • Pueden explorar amplios espacios de búsqueda.
  • Son útiles para problemas complejos donde no se conoce la forma del óptimo.

Desventajas:

  • Requieren ajustar muchos parámetros (tamaño de población, tasa de mutación, etc.).
  • Son más lentos que métodos directos como Hill Climbing.

Comparación: Hill Climbing vs Algoritmos Genéticos

Ambos algoritmos son estrategias de optimización, pero tienen enfoques diferentes.

Características Hill Climbing Algoritmos Genéticos
Estrategia Búsqueda local Búsqueda poblacional
Punto de partida Una solución inicial Varias soluciones iniciales
Exploración del espacio Muy limitada (puede quedar atrapado en óptimos locales) Amplia (mejor evita óptimos locales)
Mecanismos Evaluación y mejora incrementales Selección, cruce y mutación
Velocidad Rápido en problemas simples Más lento pero robusto
Ideal para Problemas con paisajes simples y bien definidos Problemas complejos, con muchas soluciones posibles
